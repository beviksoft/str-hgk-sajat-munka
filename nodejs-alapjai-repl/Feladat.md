### 1. Feladat
Telepített NodeJS verziószáma:
> node -v
### 2. Feladat
Telepített npm verziószáma:
> megoldás: npm -v
### 3. Feladat
Projekt függőségei:

(csak az első szint, a telepített csomagok függőségei nem)
> npm ls --depth=0
### 4. Feladat
Eslint csomag adatai:
> npm view eslint
### 5. Feladat
Elavult csomagok lekérdezése:
> npm outdated

Elavult csomagok frissítése:
> npm update
